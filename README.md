# Invoice-Generator - React App
![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB) ![](https://img.shields.io/badge/bootstrap-%23563D7C.svg?style=for-the-badge&logo=bootstrap&logoColor=white)

React-based project for creating invoices. Aimed to configure quantity, prices, tax rates, and discounts for itemised items. Save PDF versions of the invoices to your device. uses [jspdf-react](https://www.npmjs.com/package/jspdf-react) to convert data from canvas to pdf after being captured from the model.

### Live Demo
